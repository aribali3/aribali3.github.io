Covid-19 Case-Fatality Ratio by Race/Ethnicity Visualization

About

I designed, developed, and tested an information visualization solution which displayed a visualization of a COVID-19 dataset for several states and ethnicities, collected by The COVID Tracking Project.

Key Features

  • Implemented Linked List Data Structure to store and sort the data.
  • Linked List Data Structure provided sort functionality similar to insertion sort.
  • Parsed the data from a CSV file.
  • Developed JUnit Tests to throughly test the program.
  • Written in Java.
  • Code available upon request.
